## 简介

Sublime是一款常用的代码编辑器。很多Arduino爱好者使用这个软件进行Arduino的代码编写。然而第一次使用Arduino的人在如何连接Sublime和Arduino两个软件上往往一头雾水，所以这里提供一个已经打包好的Sublime + Arduino一键安装包给大家。

## 功能特性

- ✅ **开箱即用**：无需大量设置就可以直接用Sublime进行Arduino编程
- ✅ **STM32支持**：集成了Arduino for STM32补丁，可以用Arduino进行STM32编程
- ✅ **Hex文件提取**：集成了Arduino Hex补丁，可以直接提取Hex文件用于烧录
- ✅ **双版本支持**：
  - Sublime for Arduino：Sublime Text 3和Arduino分别在两个文件夹的版本
  - Sublime for Arduino Single：合并在一个文件夹的版本

## 打包内容

- **Arduino 1.8.5**
- **Ocrobot 1.6.10.0**
- **Sublime Text 3.3176**

### 包含功能

1. 使用Sublime编译Arduino的功能
2. Arduino for STM32 支持
3. Arduino 生成 Hex 补丁

## 安装说明

1. 下载对应版本的安装包
2. 解压到指定目录
3. 运行 `Install.bat` 或 `Install.vbs` 进行安装
4. 按照提示完成安装过程

## 使用方法

1. 打开Sublime Text
2. 创建新的Arduino项目文件
3. 编写代码并编译
4. 使用Hex补丁功能提取可烧录文件

## 系统要求

- Windows 操作系统
- 至少 1GB 可用磁盘空间
- 支持的Arduino板卡（Uno, Mega, STM32等）
